Another Responsive Framework

*with some extra ui goodness...

Ok, so this isn't really a framework. Not really a boilerplate. More of a, hopefully, badass starting point for creating a badass responsive website.

Responsive Framework utilizes Modernizr and Respond.js to ensure compatibility with the not so modern browser out there. Responsive Framework has been sucessfully tested in Firefox, Chrome, Safari & IE7+.

Things are still a work in progress for the time being. If you come across any bugs, or there is anything you'd like to see added feel free to submit a ticket.

Happy coding!
